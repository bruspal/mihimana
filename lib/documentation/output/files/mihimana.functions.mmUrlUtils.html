url()

url(string  $url) : string

Format the input URL to the correct format regarding mihimana.<br> if $url is absolute, nothing will be changed while relatives will be adapted to internal mihimana url<br> There is some reserved words, all of them starts with '@' <ul> <li>@home : Url for home page</li> <li>@module : the current module</li> <li>@action : the current action </li> </ul>

Parameters

the input url

useAsset()

useAsset(string  $url, boolean  $global = false,   $extension = false) : string

Generate the url to assets.

A directory called web_root/APPLICATION_assets must be set in the webserver that point to the APPLICATION/assets directory in the filesystem
On *ix system a symbolic link to APPLICATION/assets nammed APPLICATION_assets can be created in the web root directory

Parameters

the relative URI to the asset
boolean $global
