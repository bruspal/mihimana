loadHelper()

loadHelper(mixed  $helperName)

Load the helper named $helperName. First looks in the HELPERS_DIR (sits in the lib/helpers directory of the application) then in the standard MM_HELPERS_DIR for standard helpers. Throw an exception if helper not found.<br> Moreover while user helpers can be freely named in the HELPERS_DIR (but extension still .php) mihimana helpers will start by 'mmHelper'.$helperName

Parameters

mixed $helperName

a string or an array of strings helper name without extension

Throws

\mmExceptionDev
