Aide widget

Fenetre de recherche

Ce widget permet d'ouvrir une fenetre popup AJAX vers un formulaire de recherche
  • Largeur : Largeur de la zone de saisie
  • table : Table/vu de la base de donnée dans laquelle faire la recherche
  • cle : liste des champs (séparé par des ,) sur lesquel sont effectué la recherche
  • libelle : libellé des colonnes a afficher dans l'entete du tableau
  • retour : nom du champ dont la valeur renvoyé lors de la selection de la ligne
  • cols : contenu des ligne du tableau (séparé par des ,). La syntaxe '+' permet de faire des contenu complexe en concaténant les valeur de la table avec des chaine de caractere

Liste d'enregistrements

Permet de généré un tableau d'enregistrements a raison d'une ligne par enregistrement.
  • largeur : largeur des colonnes séparé par des ,
  • table : table/vue dans laquelle faire la selection
  • action : action lors d'un click sur la ligne
  • condition : condition de SQL de selection dans la table
  • lines : nombre de lignes affiché. Si il y'a plus de résultat que de ligne des bouton de navigation apparaisse
  • retour :
  • cols : liste des colonnes a affiché séparé par des ,. si la valeur est aaa:bbb aaa correspond au champ a afficher et bbb au libellé de la colonne

Select lié a un fichier

Permet de remplir une combo avec les résultats d'une recherche dans une table
  • table : table de la base de données sur laquelle appliqué la recherche
  • cle : nom de la colonne qui sera utilisé comme valeur de retour
  • Libelle : Libellé afficher dans la combo. Utilise la syntax '+'
  • condition : condition de recherche. La syntaxe est celle d'un where SQL. Pour utiliser les variables de l'ecran c'est la syntaxe $nom_var
